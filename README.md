# Menu Management Backend

Menu Management Backend is a Rails application to manage Menus and Dinners.

## How to set up locally

```
$ git clone https://github.com/matthiashaefeli/menu_backend.git
$ cd menu_backend
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
```

## Run test suit

```
$ rspec
```

### Ruby / Ruby on Rails

```
ruby 2.7.0
Rails 6.0.4.1
```
